# Gen Z Coders Frontend Foundation

### Version 1.0

---

# 1. Product Overview

## Product Name

**Gen Z Coders**

---

## Product Description

Gen Z Coders is a modern, project-based programming learning hub that transforms students from technology consumers into technology creators through structured learning journeys, real-world projects, engineering mentorship, and measurable progress.

Unlike traditional coding centers, Gen Z Coders focuses on building a **Creator Mindset**, where programming is treated as a tool for solving problems, creating products, and thinking like an engineer rather than simply memorizing concepts.

---

## Brand Slogan

> **From Consumer To Creator**
> 

---

## Vision

To become Egypt's leading Creator Mindset academy — the trusted destination for parents, students, and schools seeking practical programming education that develops the ability to build, launch, and lead.

---

## Mission

Equip students aged 8 through university level with the technical skills, engineering mindset, and creative confidence needed to move from consuming technology to creating with it through project-based learning, coaching, and measurable outcomes.

---

## Core Philosophy

Programming is **not** the product.

Programming is the tool.

The real product is the **Creator Mindset**.

Every feature, page, workflow, and learning experience should reinforce this philosophy.

---

## Core Principle

Before implementing any feature, ask:

> **Does this help students use knowledge to build something meaningful?**
> 

If the answer is yes, it aligns with the Gen Z Coders philosophy.

# 22. API Strategy

Use the real backend API whenever available.

Mocked JSON may be used temporarily during frontend development but should be replaced with real endpoints as soon as they exist.

The frontend should be built with integration in mind from the beginning.

---

# 23. Icon System

Use **Lucide React** exclusively.

# 26. Guiding Principle for Future Development

Every feature should satisfy four questions before implementation:

1. **Does it reinforce the Creator Mindset philosophy?**
2. **Does it improve trust for students and parents?**
3. **Does it provide measurable value rather than unnecessary complexity?**
4. **Can it scale with the long-term vision of Gen Z Coders without requiring architectural redesign?**
